This project implements the *Caesar Cipher algorithm* using Python.  
It allows users to *encrypt and decrypt text* by providing a message and a shift value.

---

## What is Caesar Cipher?
Caesar Cipher is a simple substitution cipher where each letter in the message is shifted by a fixed number of positions in the alphabet.

*Example:*  
- Message: HELLO  
- Shift: 3  
- Encrypted: KHOOR

---

## Features
- Encrypt text using Caesar Cipher
- Decrypt encrypted text
- Supports both *uppercase and lowercase letters*
- Non-alphabet characters (spaces, symbols) remain unchanged
- User-friendly input
